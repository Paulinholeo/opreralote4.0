### Renomeia Lote BRC - Aplicação Desktop

Aplicação GUI em Python (CustomTkinter) para renomear lotes, padronizar arquivos e analisar infrações.

#### Instalação

```bash
pip install -r requirements.txt
```

#### Execução

```bash
python main.py
```
